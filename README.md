## 🚀 核心特性 (Core Features)

*   **🚫 0 后端 (Zero Backend):** 所有业务逻辑、历史数据查询（通过 Event Log）、用户状态均在链上实时获取。前端即终端，无中心化服务器参与，彻底消除宕机与黑箱风险。
*   **⚖️ 动态锚定 (Auto-Peg):** 持仓门槛随市值自动调整。无论币价涨跌，入场门槛恒定锚定在约 **$30 USD**，确保协议的普惠性与低门槛。
*   **🎲 物理真随机 (True Randomness):** 集成最新版 **Chainlink VRF V2.5**。随机数由去中心化预言机网络生成，不可预测、不可操纵，且在链上公开可验证，确证每一份幸运皆为天意。
*   **🔓 全开源 & 永续 (OSS & Forever):** 合约与前端代码完全开源。协议不绑定特定域名，任何人都可以部署自己的 UI 界面，甚至直接在 BscScan 上与合约交互。它是真正属于社区的永续协议。

## 🕹️ 终端交互指南 (How to Play)

### 1. 激活福泽身份
在**我的运势 (Personal Terminal)** 中，应用会自动检测您的钱包余额：
*   **一键注册:** 若持仓达标（约 $30 USD 等值代币），点击“激活福泽身份”即可在链上注册。
*   **实时检测:** 余额不足时，界面将自动变更为红色预警，提醒您补充持仓。

### 2. 唤醒锦鲤 (社区悬赏)
当**下轮开奖倒计时**归零且系统显示“系统就绪”，任何人均可化身“唤醒者”：
*   **交互逻辑:** 点击卡片上的“唤醒锦鲤”按钮。
*   **福报回馈:** 成功唤醒的用户将实时获得由合约自动结算的 **BNB 赏金**（约 0.001-0.002 BNB）。

### 3. 燃料与维护 (Community Governance)
前端集成深度维护面板，确保协议自维持运行：
*   **自动转换:** 界面实时监控燃料，支持通过 **PegSwap** 一键将 BEP-20 LINK 转换为 VRF 所需的 ERC-677 LINK。
*   **异常修复:** 若链上请求超时，前端将激活“重置”功能，保障协议活跃度。

系统每 5 分钟自动检测链上市值，并根据下表调整持仓门槛。入场门槛恒定锚定在 **~$30 USD**，而满额权重门槛（获取 100% 分红权重）在市值达到 $100k 后恒定锚定在 **~$1,000 USD**。

## 📜 协议运作规则 (Protocol Rules)

| 场景 | 交互表现 | 背后逻辑 |
| :--- | :--- | :--- |
| **动态入局** | 实时余额与动态门槛对比 | 门槛随市值自动调整，始终保持约 $30 USD 价值。 |
| **线性权重** | 分红比例 (50%-100%) | 奖金比例随持仓量线性增加，达满额阈值可独揽 100%。 |
| **全自动分发** | 中奖弹窗与 BscScan 凭证 | 奖金直接 Transfer 进钱包。若失败，资金进入滞留池。 |
| **滞留补领** | 终端显示“遗落福泽” | 若自动到账失败，可在终端一键找回。 |
| **清理门户** | 锦鲤名册红字标记 | 不达标者进入 1 小时宽限期，超时可被社区清理。 |
